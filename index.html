1. Project Overview

What This Project Does

This application enables users to fill web forms using voice commands. It captures spoken input, transcribes it using OpenAI's Whisper API, extracts structured data using GPT-4o-mini, and automatically populates form fields.

Key Features

  • Voice Capture: Records audio directly from the browser microphone
  • Speech-to-Text: Uses OpenAI Whisper for accurate transcription
  • Intelligent Parsing: GPT-4o-mini extracts structured data from natural speech
  • Auto-Fill: Automatically populates form fields with extracted information
  • Manual Override: Users can edit fields before submission

Use Cases

  • Contact forms and lead generation
  • Customer support ticket submission
  • Accessibility improvements for users with disabilities
  • Mobile-friendly form filling
  • Hands-free data entry scenarios

Data Flow

  1. User clicks "Start Speaking" → Browser requests microphone access
  2. User speaks → MediaRecorder captures audio chunks
  3. User stops recording → Audio blob created
  4. Frontend sends audio to /api/transcribe endpoint
  5. Backend processes:
    • Saves audio temporarily
    • Calls Whisper API for transcription
    • Sends transcription to GPT-4o-mini with structured prompt
    • Parses JSON response
    • Cleans up temporary files
  6. Frontend receives structured data and updates form fields

Complete Server Code

server/server.js:

import express from 'express';
import multer from 'multer';
import cors from 'cors';
import fs from 'fs';
import OpenAI from 'openai';
import dotenv from 'dotenv';

dotenv.config();

const app = express();
app.use(cors());
app.use(express.json());

// Configure multer for file uploads
const upload = multer({ dest: 'uploads/' });

// Initialize OpenAI client
const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

// Transcription endpoint
app.post('/api/transcribe', upload.single('audio'), async (req, res) => {
  try {
    if (!req.file) {
      return res.status(400).json({ error: 'No audio file provided' });
    }

    const filePath = req.file.path;

    // Step 1: Transcribe with Whisper
    const transcription = await openai.audio.transcriptions.create({
      file: fs.createReadStream(filePath),
      model: 'whisper-1',
    });

    // Step 2: Extract structured data with GPT
    const completion = await openai.chat.completions.create({
      model: 'gpt-4o-mini',
      messages: [
        {
          role: 'system',
          content: `You are a precise form-data extractor. Extract the following fields from the user's speech:
- name (full name)
- email (valid email format)
- phone (phone number)
- message (general message or query)

Rules:
1. Only extract information explicitly stated.
2. If a field is missing, use null.
3. Return valid JSON only, no extra text.
4. Keep the message field concise; summarize if needed.

Example output:
{
  "name": "John Doe",
  "email": "john@example.com",
  "phone": null,
  "message": "I need help with my order."
}`
        },
        {
          role: 'user',
          content: transcription.text
        }
      ],
      temperature: 0.1,
      response_format: { type: 'json_object' }
    });

    // Parse the response
    const extractedData = JSON.parse(completion.choices[0].message.content);

    // Clean up temporary file
    fs.unlinkSync(filePath);

    // Return results
    res.json({
      transcription: transcription.text,
      fields: extractedData
    });

  } catch (error) {
    console.error('Backend error:', error);
    res.status(500).json({ 
      error: 'Processing failed', 
      details: error.message 
    });
  }
});

// Health check endpoint
app.get('/health', (req, res) => {
  res.json({ status: 'Server is running' });
});

// Start server
const PORT = process.env.PORT || 5000;
app.listen(PORT, () => {
  console.log(`Server running on http://localhost:${PORT}`);
});

6. Frontend Implementation

VoiceForm Component

client/src/components/VoiceForm.jsx:

import { useState, useRef } from 'react';
import axios from 'axios';
import './VoiceForm.css';

const VoiceForm = () => {
  const [recording, setRecording] = useState(false);
  const [processing, setProcessing] = useState(false);
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    phone: '',
    message: ''
  });
  const [transcriptionText, setTranscriptionText] = useState('');
  const [status, setStatus] = useState('');
  
  const mediaRecorderRef = useRef(null);
  const audioChunksRef = useRef([]);

  const startRecording = async () => {
    try {
      const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
      const mediaRecorder = new MediaRecorder(stream, { mimeType: 'audio/webm' });
      mediaRecorderRef.current = mediaRecorder;
      audioChunksRef.current = [];

      mediaRecorder.ondataavailable = (event) => {
        if (event.data.size > 0) {
          audioChunksRef.current.push(event.data);
        }
      };

      mediaRecorder.onstop = processRecording;
      mediaRecorder.start();
      setRecording(true);
      setStatus('Recording... Speak now.');
    } catch (err) {
      console.error('Microphone access error:', err);
      setStatus('Could not access microphone. Please check permissions.');
    }
  };

  const stopRecording = () => {
    if (mediaRecorderRef.current && recording) {
      mediaRecorderRef.current.stop();
      setRecording(false);
      setStatus('Processing speech...');
    }
  };

  const processRecording = async () => {
    setProcessing(true);
    const audioBlob = new Blob(audioChunksRef.current, { type: 'audio/webm' });
    const formData = new FormData();
    formData.append('audio', audioBlob, 'recording.webm');

    try {
      const response = await axios.post('http://localhost:5000/api/transcribe', formData, {
        headers: { 'Content-Type': 'multipart/form-data' }
      });

      const { transcription, fields } = response.data;
      setTranscriptionText(transcription);
      setFormData(prev => ({ ...prev, ...fields }));
      setStatus('Form filled automatically!');
    } catch (error) {
      console.error('API error:', error);
      setStatus('Error processing audio. Please try again.');
    } finally {
      setProcessing(false);
      audioChunksRef.current = [];
    }
  };

  const handleInputChange = (e) => {
    const { name, value } = e.target;
    setFormData(prev => ({ ...prev, [name]: value }));
  };

  const handleSubmit = (e) => {
    e.preventDefault();
    console.log('Form submitted:', formData);
    alert('Form submitted! Check the console.');
  };

  return (
    <div className="voice-form-container">
      <h2>Voice‑Enabled Form</h2>
      <p className="description">
        Speak to fill the form automatically. Say something like:
        <em>"My name is Alex, email alex@example.com, and I need a quote."</em>
      </p>

      <form onSubmit={handleSubmit}>
        <div className="form-group">
          <label>Name</label>
          <input
            type="text"
            name="name"
            value={formData.name}
            onChange={handleInputChange}
            placeholder="Will be filled by voice"
          />
        </div>

        <div className="form-group">
          <label>Email</label>
          <input
            type="email"
            name="email"
            value={formData.email}
            onChange={handleInputChange}
            placeholder="speak your email"
          />
        </div>

        <div className="form-group">
          <label>Phone (optional)</label>
          <input
            type="tel"
            name="phone"
            value={formData.phone}
            onChange={handleInputChange}
            placeholder="phone number"
          />
        </div>

        <div className="form-group">
          <label>Message</label>
          <textarea
            name="message"
            value={formData.message}
            onChange={handleInputChange}
            rows="3"
            placeholder="Your spoken message appears here"
          />
        </div>

        <div className="voice-controls">
          {!recording ? (
            <button type="button" onClick={startRecording} disabled={processing}>
              🎤 Start Speaking
            </button>
          ) : (
            <button type="button" onClick={stopRecording} className="stop-btn">
              ⏹ Stop Recording
            </button>
          )}
          {processing && <span className="processing-spinner">Processing…</span>}
        </div>

        <button type="submit" className="submit-btn">
          Submit Form
        </button>
      </form>

      {status && <div className="status">{status}</div>}
      
      {transcriptionText && (
        <div className="transcription">
          <strong>Transcription:</strong> <em>{transcriptionText}</em>
        </div>
      )}

      <div className="hint">
        Tip: Speak naturally. The AI will extract name, email, phone, and message.
      </div>
    </div>
  );
};

export default VoiceForm;

const wss = new WebSocketServer({ port: 5001 });

Validation & Error Recovery

const validateFields = () => {
  const errors = {};
  
  if (formData.email && !isValidEmail(formData.email)) {
    errors.email = 'Invalid email format';
  }
  
  if (formData.phone && !isValidPhone(formData.phone)) {
    errors.phone = 'Invalid phone number';
  }
  
  if (Object.keys(errors).length > 0) {
    setErrors(errors);
    return false;
  }
  
  return true;
};

const isValidEmail = (email) => {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
};

const isValidPhone = (phone) => {
  return /^[\d\s\-+()]{10,15}$/.test(phone);
};

13. Security Considerations

API Key Protection

  • ✅ Never expose OpenAI API key in frontend code
  • ✅ Use environment variables for sensitive data
  • ✅ Implement rate limiting on backend
  • ✅ Use API keys with restricted permissions

Data Privacy

  • ✅ Delete temporary audio files after processing
  • ✅ Implement data retention policies
  • ✅ Use HTTPS in production
  • ✅ Consider GDPR compliance for user data

Input Validation

// Add file size limits
const upload = multer({ 
  dest: 'uploads/',

// Validate file type

Rate Limiting
